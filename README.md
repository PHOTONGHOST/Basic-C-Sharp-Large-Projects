# BASIC-C-SHARP-LARGE-PROJECTS
Thie repository is for larger projects of C# and .NET Framework - Part 2 course.

**1. Method Class Assignment:**
 This is a more complex assignment for practicing void methods and named parameters.

 **2. Polymorphism Assignment:**
An assignment for practicing void and quit methods while also demonstrating polymorphism concepts.

**3. Operators Assignment:**
A more complex assignment for demonstrating overloading operators with comparison statements.

**4. ASP.NET MVC Entity Framework Assignment:**
A larger project for learning and practicing with ASP.NET MVC Entity Framework tools and applications. In this case, a basic Car Insurance web program for generating quotes
based on user input.

**5. Student Code First (Final Assignment):**
StudentCodeFirst is a demonstration of a basic Entity Framework Code-First console application, in this case, for a student name database.
